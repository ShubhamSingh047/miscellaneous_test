Frontend Devlopment

Frontend devlopment is the development of the Graphical user interface of a website, through the use of HTMLCSS and JavaScript, so that user can view and interact with that website.

  • HTML:The HyperText Markup Language or HTML is the standard markup language for documents design to be displayed in a web browser.
  • CSS: Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a documnet written in a markup language such as HTML or XML
  1. Pure Css
  2. Bootstarp (framework)
  3. Tailwind (Framework)
  4. Sass (Preeprocessor)
  • JavaScript:JavaScript offen abbreviated Js, is a programming language that is one of the core technologies of the World Wide Web, along is used to add functionality in the website.
    1. Vanilla Js
    2. jQuery (Library)
    3. React (Framework)
    4. Angular (Framework)
    5. Vue (Framework)
    6. Typescript (Preeprocessor)
